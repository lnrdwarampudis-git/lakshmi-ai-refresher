EDA Workflow ✅

  • Covered: shape, dtypes, missing, duplicates, univariate, bivariate, multivariate, target analysis, outlier summary
  • Nothing to add here — your guide is thorough

Hypothesis Testing Next priority

  • t-test, Mann-Whitney U — compare group means
  • Chi-squared — test independence of categorical features
  • ANOVA — compare 3+ group means
  • p-values, Type I / Type II errors, multiple comparisons (Bonferroni)
  • When to use each test — parametric vs non-parametric

Confidence Intervals Next priority

  • Bootstrap CIs — resampling approach, works on any statistic
  • Interpreting CIs vs p-values — what they actually mean
  • CIs for model metrics (AUC-ROC CI, accuracy CI)

Probability Distributions Medium priority

  • Normal, Poisson, Binomial, Exponential, Beta — when each appears in real data
  • scipy.stats — fitting distributions, KS test for goodness of fit
  • How distributions map to ML loss functions

A/B Testing Advanced

  • Experiment design, randomisation, control vs treatment
  • Minimum detectable effect, sample size calculation
  • Sequential testing, peeking problem, novelty effects
  • Metric selection: guardrail vs primary metric

Hyperparameter Tuning Next priority

  • GridSearchCV, RandomizedSearchCV — exhaustive vs random
  • Optuna / Bayesian optimisation — smarter search
  • Key parameters per model: n_estimators, max_depth, learning_rate, C, gamma
  • Early stopping in gradient boosting

Cross-Validation Strategies Next priority

  • Stratified K-Fold — preserve class balance in each fold
  • Group K-Fold — when rows from same entity must not split across folds
  • Time-series CV (TimeSeriesSplit) — no future leakage
  • Repeated K-Fold — reduce variance of CV estimate

Imbalanced Classes High value

  • class_weight='balanced' — free fix in sklearn
  • SMOTE, ADASYN — synthetic minority oversampling
  • Undersampling — RandomUnderSampler, Tomek links
  • Threshold tuning — optimise at the right F1/precision/recall point
  • imbalanced-learn library

Dimensionality Reduction Medium priority

  • PCA — variance explained, scree plot, how many components
  • t-SNE / UMAP — visualise high-dimensional clusters
  • When to apply: >50 features, embedding spaces, image pixels
  • Explained variance ratio — selecting n_components

Time Series Analysis Separate track

  • Stationarity, ADF test, differencing
  • Autocorrelation (ACF/PACF plots)
  • ARIMA, SARIMA, Prophet, LSTM
  • Rolling features, lag features, expanding windows
  • Train/test split must be chronological — never random

SHAP Values High ROI

  • shap library — works with any sklearn model
  • Summary plot, waterfall plot, force plot, dependence plot
  • Global vs local (per-prediction) explanations
  • SHAP vs permutation importance — when each is better

Learning Curves High ROI

  • sklearn.model_selection.learning_curve
  • Diagnosing high bias (underfit) vs high variance (overfit)
  • How many training examples do I need?
  • Validation curves — find the right regularisation strength

SQL for Analysis Essential

  • Window functions: ROW_NUMBER, RANK, LAG, LEAD, NTILE
  • CTEs and subqueries for complex aggregations
  • Date arithmetic, cohort analysis in SQL
  • CASE WHEN for feature engineering in SQL
  • Query optimisation: indexes, EXPLAIN plans

Advanced Pandas High ROI

  • groupby + agg with custom functions
  • merge types: inner, left, outer, cross — when to use each
  • pivot_table vs crosstab vs melt vs stack/unstack
  • apply vs vectorisation — performance traps
  • MultiIndex and complex hierarchical data

Chart Selection Principles Foundational

  • Which chart for which question: distribution → histogram/KDE; comparison → bar; correlation → scatter; trend → line; part-of-whole → stacked bar/pie
  • The lie factor — chart design that distorts truth
  • Pre-attentive attributes: colour, size, position
  • Storytelling: annotation, progressive reveal

MLflow & Experiment Tracking Important

  • mlflow.log_param, log_metric, log_artifact
  • Comparing runs visually in MLflow UI
  • Model registry — promoting models to staging/production
  • Integrating MLflow into your existing sklearn pipelines

Data Drift & Monitoring Advanced

  • PSI (Population Stability Index) — detect feature drift
  • KS test and Wasserstein distance for distribution shift
  • Evidently library — automated drift reports
  • Concept drift vs data drift — different causes, different fixes
