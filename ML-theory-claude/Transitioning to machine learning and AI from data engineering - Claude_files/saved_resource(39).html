Specialty Level 2+ yrs ML experience recommended ~8 weeks prep ⚠ Resources expire Mar 31 2026
Exam domains and weightings
Domain 1 (24%): Data Engineering
→ Data repositories, ingestion, transformation — overlaps heavily with DEA-C01

Domain 2 (26%): Exploratory Data Analysis
→ Feature engineering, statistical analysis, visualisation for ML

Domain 3 (28%): Modeling
→ Algorithm selection, training, hyperparameter tuning, evaluation — deepest ML content

Domain 4 (22%): ML Implementation and Operations
→ SageMaker deployment, inference types, MLOps, security
MLS-C01 vs MLA-C01 — the real difference
MLA-C01 (Associate): How to implement and operationalise ML on AWS. SageMaker-heavy. Practical engineering focus. "How do you configure X to achieve Y?"

MLS-C01 (Specialty): Deeper ML theory. Algorithm selection logic. Hyperparameter impact on model behaviour. Architecture design. "Why would you choose XGBoost over linear learner for this use case?"

Domain 3 (Modeling, 28%) is what makes MLS-C01 genuinely hard — it requires you to know ML algorithms deeply, not just know which AWS service to use.
Is MLS-C01 worth it for you?
✓ Highest prestige of all 4 ML-related AWS certs

✓ Domain 3 (Modeling) rewards your Phase 1 ML study directly
✓ Completes the full AWS ML certification stack

YOUR ORDER + TIMELINE
Personalised Certification Timeline
Mapped to your 8-week ML study plan — exactly when to attempt each cert
The golden rule: Never take a cert as a study milestone — take it as a validation of work you have already done. Each cert below is scheduled AFTER you have the practical skills it tests, not before. Studying for a cert before you have the experience means memorising without understanding.
The complete timeline
Month-by-month breakdown
Month 1 — DEA-C01 (while starting Phase 1 ML study)
Weeks 1-2: Identify your gaps. Buy Stephane Maarek's DEA-C01 course on Udemy (~$15 on sale). Watch sections on Lake Formation, MWAA, and Glue DataBrew — these are likely your real gaps. Everything else is review.
Weeks 3-4: Do 2 full Tutorials Dojo practice exams. Score 80%+ on both before scheduling. Take the exam. Pass.

Why it works: Your existing AWS DE skills cover ~80% of this exam. You are filling 4-5 specific knowledge gaps, not learning everything from scratch. Treat it like a gap-filling exercise, not a full certification course.

Parallel activity: Start Phase 1 Week 1 (linear algebra) simultaneously. The DEA study does not interfere with ML study.
Month 2 — AIF-C01 (while doing Phase 1 Weeks 3-4)
Weeks 5-6: AIF-C01 prep runs PARALLEL to Phase 1 Week 3 (ML pipeline) and Week 4 (overfitting). The ML concepts you are studying for your learning plan map directly onto AIF-C01 Domain 1. You are essentially double-dipping — ML study IS AIF-C01 exam prep.
Weeks 7-8: Focus AIF-C01 prep on GenAI/Bedrock domain (Domain 3) — this is the part your ML study does NOT cover. Read the AWS Bedrock docs. Do the free AWS Skill Builder AIF-C01 course.

Why now: $75 cert, fast to prep, GenAI domain is directly relevant to your Azure OpenAI work. The responsible AI section reinforces good habits for production ML work.
Months 3-4 — Phase 1 ML Study (no exams, just build skills)
Weeks 9-16: Complete Phase 1 Weeks 5-8. Train real models. Build the Databricks capstone project. Finish the IBM Telco Churn project. Get your first model into MLflow Model Registry.

Why no cert this period: This is your skill-building period. The MLA-C01 requires real hands-on experience — you need to have actually done the things it tests, not just read about them. Two months of focused ML project work is the fastest MLA-C01 prep you can do.

Parallel activity: Create a free AWS account and start hands-on SageMaker labs. AWS Skill Builder has free SageMaker courses. This gives you the SageMaker-specific knowledge MLA-C01 tests.
Month 5 — MLA-C01 (after Phase 1 complete + SageMaker hands-on)
Weeks 17-20: Dedicated MLA-C01 prep. Take Stephane Maarek + Frank Kane's "AWS Certified ML Engineer Associate: Hands On!" course (Udemy). Do SageMaker Studio labs for each domain — especially SageMaker Pipelines and Model Monitor. These two are consistently flagged as hardest domains.
Week 21: 3 full practice exams (Tutorials Dojo). Score 82%+ before scheduling — MLA-C01 is harder than DEA-C01 and the passing bar rewards more preparation.

Why it clicks now: You have Phase 1 done. Feature engineering, model evaluation, bias-variance, MLflow — all familiar from your study. MLA-C01 Domain 1 (data prep for ML) maps directly to Weeks 7-8 of your study plan. You are validating skills you actually have.
Months 6-7 — Phase 2 ML Study (Deep Learning + PyTorch)
Weeks 22-32: Phase 2 — CNNs, RNNs, Transformers. Build real deep learning projects. This gives you the neural network depth that MLS-C01 Domain 3 (Modeling) tests.
Month 8 — MLS-C01 (if pursuing — ⚠ prep resources expire Mar 31 2026)
Weeks 33-40: MLS-C01 is your optional final boss. If you are targeting senior ML Engineer or ML Architect roles, this cert is worth the $300 and 8 weeks. If you are satisfied with MLA-C01 and your Databricks ML Associate cert, you can skip MLS-C01.

Decision rule: If your target role at your company requires AWS-specific ML architecture decisions (choosing SageMaker vs EMR vs Bedrock for different use cases), do MLS-C01. If your primary platform is Databricks/Azure and AWS is secondary, MLA-C01 is sufficient.

⚠ Action required: Current exam prep resources (official practice questions, study guide) expire March 31, 2026. Start MLS-C01 prep no later than January 2026 if you want to use current materials.

DEA-C01 — 3-week study plan
Week 1 — Gap identification and targeted study
Day 1: Take a free practice test cold (AWS official sample questions). Note every wrong answer — these are your gaps.
Days 2-5: Study ONLY your gap areas. Typical gaps: Lake Formation, MWAA, EMR variants, Redshift Serverless.
Days 6-7: Read the official exam guide PDF cover to cover. Tick off every service you know. Study the ones you don't.
Week 2 — Practice exams + targeted review
Days 8-10: First full Tutorials Dojo practice exam (65 questions, timed). Review every wrong answer — read the explanation AND the AWS docs page for the service.
Days 11-13: Second full practice exam. Target 80%+ before moving to week 3.
Days 14: Review weakest domain only.
Week 3 — Final prep and exam
Days 15-18: Third practice exam. Review. Build a cheat sheet: one page per domain with key decision rules (e.g. "Kinesis Streams vs Firehose: if you need custom processing → Streams; if you just need delivery → Firehose").
Day 19: Schedule and take the exam. Morning slot preferred — sharper focus.

MLA-C01 — 6-week study plan
Weeks 1-2 — SageMaker foundations
Take Maarek + Frank Kane's MLA-C01 course. Focus on SageMaker Studio, Training Jobs, and Processing Jobs. For each SageMaker service — actually launch it in your AWS account and run a small job. You cannot pass MLA-C01 without hands-on SageMaker experience.
Weeks 3-4 — Deployment and monitoring
Deep dive Domain 3 (deployment) and Domain 4 (monitoring) — consistently reported as hardest. Build a SageMaker Pipeline end-to-end: data preprocessing → training → evaluation → deployment. Set up Model Monitor on a live endpoint. This hands-on work IS the exam prep.
Weeks 5-6 — Practice exams and edge cases
3 full Tutorials Dojo MLA-C01 practice exams. Score 82%+ before scheduling. Pay attention to scenario-based questions — MLA-C01 asks "given THIS business requirement and THESE constraints, which SageMaker configuration do you choose?" Pattern recognition here is everything.

MLS-C01 — 8-week study plan (if pursuing)
Weeks 1-2 — ML algorithm review
MLS-C01 Domain 3 (Modeling, 28%) is the differentiator. Review your Phase 1 notes: linear regression, logistic regression, decision trees, random forest, XGBoost, SVM, k-means. For each: when to use it on AWS, which SageMaker built-in algorithm maps to it, what hyperparameters matter and why.
Weeks 3-4 — Deep learning and NLP
MLS-C01 tests neural networks, CNNs, RNNs, and NLP — this is Phase 2 content. Work through these with SageMaker: image classification with SageMaker's built-in image classifier, sequence-to-sequence, BlazingText for NLP. The exam tests which SageMaker algorithm to use for which problem.
Weeks 5-6 — Data engineering + EDA domains
Domains 1 and 2 overlap heavily with DEA-C01 — these should be fast. Focus on the ML-specific parts: feature engineering for ML, handling imbalanced datasets, EDA for model selection, statistical significance tests for feature selection.
Weeks 7-8 — Practice exams and scenario mastery
4 full practice exams (Tutorials Dojo + AWS official). MLS-C01 has a higher passing score (750 vs 720) and more abstract scenario questions. Target 85%+ on practice exams before scheduling. The extra 5% buffer matters here.
