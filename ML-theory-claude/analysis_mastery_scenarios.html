Missing value imputation
Age is 20% missing. Learn median imputation + KNN imputation. Add a binary missing indicator column — whether age was missing may itself be predictive.
Categorical encoding
Sex (binary encode), Embarked (one-hot), Pclass (ordinal). Practice every encoding type on one dataset.
Feature engineering
Create FamilySize = SibSp + Parch + 1, IsAlone, extract Title from Name (Mr/Mrs/Miss/Master), FarePerPerson. These boost accuracy significantly.
Class imbalance handling
62% died vs 38% survived — mild imbalance. Learn class_weight='balanced', why accuracy is misleading here, and use AUC-ROC + F1 instead.
Model interpretability
Use SHAP to answer "what factors most predicted survival?" — Sex, Pclass, Age, FamilySize will dominate. Great first SHAP exercise.
Model comparison

from sklearn.model_selection import train_test_split, cross_val_score, StratifiedKFold from sklearn.ensemble import GradientBoostingClassifier from sklearn.metrics import average_precision_score, f1_score from imblearn.over_sampling import SMOTE import pandas as pd cat_cols = df.select_dtypes('object').columns df_enc = pd.get_dummies(df, columns=cat_cols, drop_first=True) X = df_enc.drop('Churn', axis=1).astype(float) y = df_enc['Churn'] X_tr, X_te, y_tr, y_te = train_test_split(X, y, stratify=y, test_size=0.2, random_state=42) results = {} clf_base = GradientBoostingClassifier(n_estimators=100, random_state=42) clf_base.fit(X_tr, y_tr) results['No adjustment'] = average_precision_score(y_te, clf_base.predict_proba(X_te)[:,1]) clf_w = GradientBoostingClassifier(n_estimators=100, random_state=42) X_tr_w, y_tr_w = X_tr.copy(), y_tr.copy() smote = SMOTE(random_state=42) X_sm, y_sm = smote.fit_resample(X_tr, y_tr) clf_w.fit(X_sm, y_sm) results['SMOTE'] = average_precision_score(y_te, clf_w.predict_proba(X_te)[:,1]) print("Average Precision (AUC-PR) — better than AUC-ROC for imbalanced:") for name, score in results.items(): print(f" {name:20s}: {score:.4f}")
Why AUC-PR not AUC-ROC here? AUC-ROC can look great (0.85) even when a model ignores the minority class. AUC-PR (Average Precision) punishes models that miss positive cases — it is the right metric when the minority class is what you care about.
